## 0. 竞争现状判断（基于 2026-07 虾皮台湾「按壓旋轉球」类目实拍）

| 观察 | 事实 | 对我们的含义 |
|---|---|---|
| 价格带 | NT$80–230，普遍标「5折」「4.5折」 | **不打价格战**。$80 的货是白牌走量逻辑，跟进只会亏 |
| 主图风格 | 大红大紫促销风、爆炸贴纸、买一送一 | 全场都在喊，**安静反而显眼**。黑底琥珀主图在信息流里是差异化武器 |
| 卖点话术 | 「解壓神器」「舒壓小物」「辦公室療癒」 | 关键词已被教育，SEO 标题可以直接借用这些词 |
| 「3D列印」标签 | 已有卖家当卖点用 | 3D 打印本身不再是稀缺卖点，**要卖的是设计、配色、手感、品牌** |
| 卖场评价 | 新上架、5.0 分、>7 天到货 | 市场是新的、竞争者也是小卖家，早期占位窗口还在 |

**核心结论：同一颗球，白牌卖 NT$80 是「玩具」，我们卖 NT$249–320 必须是「桌面好物」。差价靠三件事撑起来：视觉（配色/摄影/包装）、内容（制作故事/手感表达）、品牌（客林玩好物账号背书）。**

## 1. 总体架构：一次拍摄，全渠道分发（内容工厂模型）

```
                ┌─────────────────────────────┐
                │   母素材（每 SKU 拍一次）      │
                │   9 组镜头 + 1 条 30s 母视频   │
                └──────────────┬──────────────┘
        ┌──────────┬──────────┼──────────┬──────────┐
        ▼          ▼          ▼          ▼          ▼
     自建站      虾皮TW      小红书    TikTok/Reels  FB/IG贴文
    (品牌资产)   (成交)     (种草+成交)   (引流)      (信任+再营销)
        ▲                                │
        └────────── bio 链接全部回流 ─────┘
```

**渠道分工（不要让每个渠道都干所有事）：**

| 渠道 | 角色 | 北极星指标 | 转化动作 |
|---|---|---|---|
| 自建站 | 品牌总部：全价成交、故事、邮件订阅、SEO | 邮件订阅数 / 直购订单 | Stripe Checkout |
| 虾皮 TW | 台湾履约 + 平台信任（评价、免运门槛） | 订单量 + 卖场评分 | 平台内下单 |
| 小红书 | 种草 + 大陆成交（挂车/私域） | 收藏率 + 进店率 | 小红书店铺/私信 |
| TikTok | 纯引流：satisfying 短视频 | 完播率 + 主页点击 | bio → 自建站/虾皮 |
| IG / FB | 品牌橱窗 + 再营销素材池 | 关注增长 + 链接点击 | bio link / Shop 标签 |

**铁律：所有平台 bio 链接统一指向自建站（或 link-in-bio 页），自建站再分流到虾皮/小红书。品牌资产必须沉淀在自己手里（邮件列表 + 域名）。**

## 2. 视觉包装系统

### 2.1 品牌视觉直接复用（globals.css token）

- 背景 `#0A0A0A` 深炭灰 · 文字 `#ECEAE5` 暖白 · 强调 `#F2A93B` 暖琥珀 · 金属 `#2E2E33`
- 调性：暗调工业 + 单一暖色光源 + Bauhaus 几何 + curated minimalism

### 2.2 每 SKU 固定拍摄清单（母素材 9 组镜头）

| # | 镜头 | 用途 |
|---|---|---|
| 1 | 黑底 + 单侧暖光的「英雄照」 | 自建站 hero / IG / 虾皮第 2 图 |
| 2 | 浅底（水泥灰/米白）干净产品照 | 虾皮主图底图 / 白底需求 |
| 3 | 手持比例照（掌心/指尖） | 尺寸传达，全渠道通用 |
| 4 | 动作特写：按下去的瞬间 | 短视频封面 / 详情页 GIF |
| 5 | 旋转中的长曝光或慢门 | 「会动」的证明，TikTok 封面 |
| 6 | 细节纹理微距（3D 打印层纹） | 「手作感」叙事，小红书 |
| 7 | 桌面场景照（键盘/笔记本旁） | 场景种草，FB/IG |
| 8 | 拆件/结构展示 | 「可拆可换」品牌主张 |
| 9 | 包装 + 产品合照 | 开箱预期管理，虾皮末图 |
| 视频 | 30s 母视频：按压→旋转→停下→logo | 剪出 6s/15s/30s 三个版本 |

### 2.3 实体包装（低成本高感知）

- 牛皮纸盒或磨砂黑盒 + **琥珀色 `#F2A93B` 贴纸/腰封**（一眼认出品牌）
- 盒内一张小卡：一句品牌话 + 二维码（扫码回自建站 → 订阅邮件领下次折扣）
- 卡片背面：「小批量 3D 打印，每一件有细微层纹」——把工艺特征提前变成预期，差评防御

## 3. 各渠道打法与模板

### 3.1 自建站（品牌总部）

- 产品页已有 MDX 结构：`## 它适合谁` / `## 手感` / `## 下单前知道`——保持这个「说人话」结构
- 每个 SKU frontmatter 填满 `purchaseLinks`（shopee / xiaohongshu / instagram / direct），详情页展示「在你习惯的平台购买」
- 首页 hero 用镜头 1（黑底琥珀英雄照），Journal 用来发工坊日记（= 小红书内容的长文版，SEO 沉淀）

### 3.2 虾皮台湾（成交主力）

**标题公式（繁体，堆满关键词但前 20 字讲人话）：**

```
按壓旋轉球 指尖解壓球 舒壓小物 辦公室療癒 3D列印設計款 台灣出貨 OneGoods 玩物工坊
```

**主图（第 1 图）公式：** 浅底大产品图（占 70%）+ 左上一句卖点（「一按即轉 · 安靜解壓」）+ 右下小徽章（設計款/現貨）。**不要**用爆炸贴纸和大红底——全类目都在喊，干净就是差异化。

**图 2–8 顺序：** ② 黑底品牌英雄照 → ③ 手持比例 → ④ 按压动作 → ⑤ 配色全家福 → ⑥ 细节纹理+「手作说明」 → ⑦ 场景照 → ⑧ 包装照。

**定价：** 锚定 NT$320 划线价，日常 NT$249–280；不做 5 折，用「兩件 9 折」「滿 $399 贈貼紙」代替降价。竞品 $80 的存在反而帮我们分层——详情页第一屏就回答「为什么值三倍」：设计配色、打磨手感、品牌包装、台湾现货。

### 3.3 小红书（种草 + 客林玩好物账号协同）

三类内容轮流发（对齐母账号战略 v1.1.3 的选题池）：

1. **工坊过程**：打印延时、失败品、调色过程——「诚实的制作过程」是白牌做不到的内容
2. **手感实拍**：第一视角按压/旋转 + 真实环境音（ASMR 向）
3. **场景种草**：工位美学、「给桌面留一个小暂停」情绪文案

标题模板：`打了 47 版才调出这个蓝｜3D 打印解压球` / `工位上最没用但最常摸的东西`。评论区置顶导购链接。

### 3.4 TikTok / IG Reels（引流）

- 内容 = 6–15s 循环：**按压声 + 旋转 + 突然停下**（satisfying/oddly satisfying 标签流量池）
- 三个固定系列：`#satisfying 旋转循环`、`3D printing timelapse`、`desk setup 场景`
- 文案极简，英文为主（TikTok AU/TW 都吃英文 tag）：`printed, assembled, spun. #fidget #3dprinting #desksetup`
- bio 链接 → 自建站

### 3.5 Facebook / IG 贴文（信任 + 再营销）

- IG grid 只发黑底琥珀调性的图（橱窗逻辑，九宫格整齐）
- FB 主要服务台湾客群：虾皮活动同步、买家秀转发
- 后期投放时，这两个号的素材池就是再营销广告素材

## 4. 每周内容节奏（一人可执行的最小闭环）

| 日 | 动作 | 产出 |
|---|---|---|
| 周一 | 拍摄日：一个 SKU 过一遍 9 镜头清单 | 母素材入库 `public/images/` |
| 周二 | 剪辑：母视频 → 6s/15s/30s | 3 条短视频 |
| 周三 | 发布：TikTok + Reels（同素材） | 2 平台各 1 条 |
| 周四 | 小红书图文 1 篇（过程或手感） | 1 篇笔记 |
| 周五 | 自建站 Journal 长文版 + 虾皮卖场维护（回评/上新） | SEO 沉淀 |
| 周末 | 看数据：完播率/收藏率/进店率，定下周选题 | 迭代 |

**原则：一周只围绕一个 SKU 做内容，打透再换。**
